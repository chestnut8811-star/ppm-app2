1本ツールの概要

本ツールは、大動脈内バルーンパンピング (IABP) の拡張 (Inflation) / 収縮 (Deflation) タイミングを、ブラウザ上で安全に学習するためのシミュレーターです。実機器を触る前の予習、ICU 配属時の復習、CE / Ns / 医師の自己学習にお使いいただけます。

本ツールで学べること

  • 正常な ECG / AP / バルーン内圧 の波形パターン
  • ECG・圧・ペーシング・インターナルの4 つのトリガーモードの違い
  • 早期 / 遅延 の拡張・収縮それぞれの波形変化と臨床的影響
  • 補助拡張ピーク (C) / BAEDP (E) / 補助 SBP (D) などの血行動態指標
  • 不整脈 (PVC、Af) が混入したときの読み方
⚠ 重要: 本ツールは個人が作成した教育用シミュレーションです。波形は学習用に単純化されており、実機・実症例の正確な再現ではありません。臨床判断は必ず 担当医師・院内ルール・機器マニュアルを優先してください。

2画面の見方

3 つの波形

画面中央の黒いモニター領域に、ICU で見るのと同じ 3 波形が縦に並んで表示されます (スイープ式: 時間軸固定、上端の三角カーソルが左→右へ走査)。

波形色意味
ECG緑心電図 (P-QRS-T)。R波がトリガーに使われる。
AP (カテ先端圧)赤動脈圧波形。ディクロティックノッチ、補助拡張ピーク、BAEDP を観察。
バルーン内圧青バルーン内のヘリウムガス圧。プラトーは AP 拡張期ピークと整合するのが正常。

縦線マーカー

  • Inf (オレンジ): バルーン拡張開始タイミング
  • Def (ピンク): バルーン収縮開始タイミング
  • 背景の細い縦線: 各R波の位置
  • オレンジ半透明シェード: バルーン拡張中の期間
  • 黄色の点: モード固有の参照点 (T波終末・ノッチ底・ペーシングスパイク)

左サイドバー (操作パネル)

トリガーモード
ECG / 圧 / ペーシング / インターナル の 4 択
動作設定
心拍数 (40-140 bpm)、補助比 (1:1, 1:2, 1:3)、一時停止、ビープ音 ON/OFF とピッチ
タイミング調整
Inflation %、Deflation %、5 種のプリセット (正常 / 早期拡張 / 遅延拡張 / 早期収縮 / 遅延収縮)

右ペイン (情報パネル)

  • タイミング評価: 拡張・収縮の判定 (正常 / 警告 / 重度)
  • 血行動態指標: A〜E の数値および3つの判定基準
  • 解説: 選択中モードの臨床ポイント

34 つのトリガーモード

モード トリガー源 適応
ECG R波検出 洞調律・ECG良好な症例 (最も一般的)
圧 動脈圧のディクロティックノッチ検出 電気メス使用中、ECG不良時、心房細動 など
ペーシング V-pace スパイク検出 100% ペーシング依存例 (完全房室ブロック等)
インターナル 固定レート (40-120 bpm) 心停止、VF、無脈性 VT、CPB 中

ECG / 圧 / ペーシングモードの違いを観察するコツ

  • ECG モード: ECG 上に "T波終末" の黄色マーカーが出る → そこが拡張目標
  • 圧モード: AP 上に "ノッチ底 (AVC)" の黄色マーカーが出る
  • ペーシングモード: ECG 上にペーシングスパイク (鋭い縦線) → そこがトリガー
  • インターナルモード: ECG は VF (カオス) または心静止 (フラット)、AP は balloon driving のみ
ヒント: 「ECG ノイズが多くて R波が拾えない」状況を模擬したい時は 圧モードに切り替えてみましょう。ノッチ位置を基準にトリガーが入る挙動が確認できます。

4タイミング調整 (Inflation / Deflation)

スライダーの仕組み

スライダーは 心周期 (cycle) の何 % の位置でバルーンを動作させるかを指定します。心拍数を変えても比率は維持され、自動で絶対 ms に換算されます。

  • Inflation 目標: 35% (60bpm なら R波後 350 ms ≒ T波終末・大動脈弁閉鎖)
  • Deflation 目標: 95% (60bpm なら R波後 950 ms ≒ 次R直前・等容収縮期)

4 つの典型的タイミングエラー

エラー 波形特徴 臨床影響
早期拡張
(Early Inflation)		 ノッチ前で拡張、V字消失、補助拍 SBP 上昇 AV未閉鎖中に balloon が膨らみ後負荷上昇 → 心筋酸素消費増加・心拍出量低下
遅延拡張
(Late Inflation)		 ノッチが分離して見え、augmentation 不十分 拡張期 augmentation 効果が低下し、冠動脈灌流改善が不十分
早期収縮
(Early Deflation)		 BAEDPに達した後 AP が再上昇 (U字) 後負荷軽減効果なし、冠動脈・脳血管への逆行性血流リスク
遅延収縮
(Late Deflation)		 次の収縮期にバルーン残存、補助拍 SBP 上昇 後負荷増加で心仕事量増大、有害

左サイドバーの「プリセット」ボタンで各エラーを即座に再現できます。

注意: プリセットは波形の "見た目の違い" を確認するためのもので、実機の安全装置を再現していません。

5評価とランドマーク (A〜E)

5 つのランドマーク

AP 波形上に「波形ランドマーク (A-E) を表示」がオンの時、以下の点に白丸+黄色ラベルが付きます。

記号名称典型値 (mmHg)
A非補助収縮期圧120
B非補助拡張末期圧 (EDP)78
C補助拡張期ピーク (Diastolic Augmentation Peak)145
D補助収縮期圧 (Assisted Peak Systolic)115
EBAEDP (Balloon Aortic End Diastolic Pressure)62

3 つの判定基準

右ペインの「血行動態指標」下に、補助効果の良否を判定する 3 行が表示されます。

  • C ≧ A: 補助拡張期ピーク ≧ 非補助収縮期圧 → 拡張期 augmentation が有効
  • D ≦ A − 5: 補助 SBP が 5 mmHg 以上低下 → 後負荷軽減効果あり
  • E ≦ B − 15: BAEDP が 15 mmHg 以上低下 → 効果的な deflation

すべて緑(✓)になっていれば、補助 IABP が血行動態的に最大効果を発揮している状態です。

6テスト機能

テストヒント (なし / あり)

「学習」カードで「テストヒント なし/あり」を選んでからテストを開始してください。なしの場合、答えを示唆する要素 (T波終末マーカー・ノッチマーカー・拡張帯シェード・ランドマーク A-E) はすべて非表示になります。

判定テスト (5 択)

「テストモード開始」をクリックすると、Inflation / Deflation がランダムにずらされた波形が出題され、5 択 (正常 / 早期拡張 / 遅延拡張 / 早期収縮 / 遅延収縮) で判定します。

  1. 波形を観察
  2. 5 択ボタンから 1 つ選んでクリック
  3. ◯ / × と解説が表示される
  4. 「次の問題」で次へ

難易度

難易度調律PVC 頻度
易 (緑)洞調律のみ0%
中 (黄)洞調律+散発PVC~25%
難 (赤)Trigeminy + 連発PVC~50%
鬼 (紫)Af (P波消失+細動波) + PVC20% PVC + 80% Af

調整テスト (実技形式)

「調整テスト開始」では、波形を見ながら Inflation / Deflation スライダーを動かして自分で最適タイミングに合わせる練習ができます。

7ビープ音

  • 音声 ON を一度クリックして Web Audio を初期化します (ブラウザのセキュリティ仕様)
  • ピッチスライダーで 800〜5000 Hz の好みの音程に
  • R 波検出ごとに短い高音ビープ
  • VF モード時のみ、ECG ノイズに反応して不規則ビープ ("ピッピピピ、ピ、、、ピッピピピピピ")
  • 心静止モードは無音 (実機 asystole alarm は再現していません)
iPhone の注意: マナーモード (サイレントスイッチ) でも本ツールは音を出そうとしますが、iOS の仕様で再生されないことがあります。スピーカーモードで一度画面をタップしてから「音声 ON」を押してください。

8スマートフォンでの使い方

  • レイアウトは縦 1 カラムに自動切り替え
  • タップターゲットは 44px 以上を確保 (Apple HIG 推奨に準拠)
  • 波形の細部を見たい時は横向き (ランドスケープ) 表示が見やすいです
  • ホーム画面に追加すると、URL バーが消えてフルスクリーン風に使えます
    iOS Safari → 共有 → ホーム画面に追加 / Android Chrome → メニュー → ホーム画面に追加
音声テスト手順 (スマホ): ① 画面を一度タップ → ② 「音声 ON」 → ③ ピッチを調整 → ④ R 波ビープを確認

9学習のすすめ方

初学者向け 30分コース

  1. 「ECG トリガー」+「1:2 補助」+「正常プリセット」で 1 分観察 — まず正常波形を覚える
  2. プリセット「早期拡張」「遅延拡張」「早期収縮」「遅延収縮」を順番にクリック — 各エラーの見た目を覚える
  3. スライダーを動かして自分で正常に戻してみる
  4. HR を 90 / 120 bpm に変えて、ノッチ位置の変化を観察
  5. テストモード (易) で 5 問挑戦
  6. 解説を読みながら本ツールに戻って波形を再確認

中級者向け

  1. 圧トリガーモードに切替 — ECG が信頼できない状況をシミュレート
  2. テスト難易度「中」で PVC 混入時の判定練習
  3. 1:1 / 1:3 補助比を試して波形差を観察
  4. 「鬼」(Af) モードで Af + PVC を判定 — 実臨床に近い難しさ

10FAQ / トラブルシューティング

波形が動かない

左サイドバーの「一時停止」が押されていないか確認 → 「再開」をクリック。

音が鳴らない

  • 「音声 ON」を 1 回クリックして Audio を初期化
  • ブラウザのタブミュート (タブを右クリック → 「サイトのミュート解除」)
  • スマホはサイレントモード解除 + 画面タップ → 音声 ON
  • VF/心静止 モードでは R 波がないため通常ビープは鳴りません

テスト中にスライダーや評価表示が消える

仕様です。テストヒント「なし」を選択すると、答えを示唆する UI が自動非表示になります。テストを終了 (「テスト終了」) すれば元に戻ります。

波形のディクロティックノッチが見えにくい

HR が高すぎ (120 bpm以上) ると 1サイクルの幅が狭くなり詳細が見えにくくなります。HR を 60-80 bpm に下げて観察してください。

スマホで波形が小さい

横向きにすると波形エリアが広がります。または、波形領域をピンチアウトしてズーム (一部ブラウザのみ対応)。

iPhone Safari で音声 ON にしても音が出ない

iOS は厳しい自動再生制限があります。

  1. ページのどこかを 1 度タップ
  2. 「音声 ON」ボタンをタップ (このタップが「ユーザー操作」と認識される必要があります)
  3. サイレントスイッチ (本体側面) がオフ (= 着信音ありモード) になっているか確認
  4. 音量ボタンで上げる

このツールの波形は実機と完全に同じですか?

いいえ。波形は学習用に単純化された理論モデルです。実際の症例では、循環状態・カテーテル位置・機種によって波形は大きく異なります。実臨床では必ず実機のマニュアル・施設の手順書を参照してください。

⚠ 最後にもう一度: 本ツールは個人が作成した教育・学習補助目的のシミュレーションであり、診療判断・治療判断を代替するものではありません。実臨床では、患者状態・使用機器・院内運用・主治医の指示等により対応が異なる場合があるため、必ず担当医師の指示および施設の院内ルール・機器マニュアルを優先してください。
