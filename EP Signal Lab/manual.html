1. これは何か

カテーテルアブレーション(EP検査)で用いる体表心電図と心内心電図(EGM)を、実患者データを使わずにすべて合成波形でリアルタイム表示する教育ツールです。臨床工学技士・アブレーション業務に入る新人スタッフ・EP検査の基礎を学ぶ方を対象に、疾患ごとの「電位の時間関係」(どの電極がどの順で・どれくらいの間隔で興奮するか)を学べます。

外部ライブラリやインターネット接続は不要です。index.html 1ファイルで完結します。

2. 画面の構成

  • 上部ヘッダー:タイトルと注意書き。
  • 左サイドバー:すべての操作(疾患選択・速度・ゲイン・各機能)。
  • 中央の波形表示:黒背景に上から各チャネルが流れる。左端にチャネル名、上端に時刻、右上に現在の設定(HUD)。
  • 右パネル:選択中の疾患の解説・自動計算した interval・チェック結果・やさしい解説・用語ガイド。
  • 下部:免責(常時表示)。

4. 波形の見方(中央)

チャネル(電極)

左端に固定表示。上から「体表心電図(ECG II・V1)」→「心内カテーテル(HRA・His・CS・RVA・ABL 等)」の順に並びます。疾患により構成が変わります。

イベントラベル

重要な電位に小さなラベルが付きます(ラベル ON 時):

  • P 体表のP波(心房の興奮) / A 心房電位 / H His束電位 / V 心室電位 / PV 肺静脈電位
  • 正常(洞調律)では His の行で A→H→V の順に並びます。これが基本。

グリッドと時刻

細かいマス目 + 太い秒線。上端の -1s / -2s … は「今(右端)」からどれだけ前か。波形とグリッドは常に一致します。

右上の表示(HUD)

現在の Sweep / EGM gain / 表面gain / ノイズ / FPS(描画の滑らかさ)を表示。

時間差を細かく見たいときは Pause → 計測カーソルON → 2点クリックで ms 計測、が基本の流れです。

5. 解説パネル(右)

  • 疾患名・概要:選択中の疾患の説明。
  • やさしい解説(緑枠・黄色フラグ):素人向けに「これは何?/心臓で起きていること/見るところ/治療の考え方」を平易に。「AI下書き・臨床未監修」と明示。
  • 重要 interval(自動計算):CL / AH / HV / VA / AA / VV をミリ秒で自動表示。
  • 不変条件チェック:合成された波形から AH・HV・興奮順序などを実測し、標準範囲に入っているかを ✓/✗ で表示(教材としての自己点検)。
  • 見るポイント / 典型順序 / 鑑別 / ターゲット / 危険 / 初学者の誤り:標準的なEP生理に基づく解説文。
  • 用語ガイド:A波・CL・VA・HRA・CS など略語の意味を平易に。
interval の数値や解説文は、画面下端・右下にあるとおり臨床監修待ちのドラフトです。教材として確定する前に院内の責任者(指導医・主任技士)の確認を受けてください。

モード内容

🎮 ゲーム
(診断チャレンジ)		ランダムに頻拍を出題(疾患名は伏せられる)。制限時間内に6択で回答し、速さと連続正解でボーナス加点。全5問でランク判定(研修中〜達人)。ハイスコアを保存。難易度=易しい/普通/難しい。

7. 音声(EGM 1ch 聴診)

サイドバー下部の「音声」で ON にし、聴きたい1チャネルを選ぶと、その電位(A/H/V/PV など)が画面の「今」を通過するたびに高音のクリック音で鳴ります。EPラボで術者がマッピング電極の音を「聴いて」早期で鋭い局所電位かを判断するのと同じ使い方です。

  • 既定の聴診chはマッピング向き(ABL → His → RVA の優先)。音量はスライダーで調整。
  • 全チャネル同時には鳴りません(雑音化するため。実臨床でも1chを選んで聴きます)。
  • 再生中のみ発音します。最初の「音声ON」操作で音が有効になります(ブラウザの仕様)。

8. 疾患一覧

疾患ひとことで
Sinus rhythm /
Baseline EPS		正常な洞調律。A→H→V の基本を確認する「お手本」。
AVNRT slow-fast房室結節近くで電気が小さく旋回。AとVが近接、VAが短い。
Orthodromic AVRT余分な「抜け道(副伝導路)」を逆走する頻拍。VAがAVNRTより長め。
Typical atrial flutter心房を大きく旋回。ノコギリ歯状の連続波。峡部(CTI)が的。
Focal atrial tachycardia心房の一点が高速発火。最早期(ABL distal)が治療の的。
Atrial fibrillation / PVI心房が不規則に細動。肺静脈の隔離(PVI)が柱。PV電位を見る。
PVC / Idiopathic VT心室から早期の余分な拍。起源の局所Vが体表QRSより先行。
