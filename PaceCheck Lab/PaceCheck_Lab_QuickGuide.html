Quick Guide

CIED check Lab 簡易取扱説明書

植え込み型心臓電気デバイス（CIED）のチェック業務を、教育・記録・操作練習の面から支援するブラウザ用ツール群の使い方をまとめた簡易ガイドです。

⚠ ご注意：本ツールは個人で作成した教育・確認補助ツールであり、診療判断を代替するものではありません。実臨床では、患者状態・使用機器・院内運用・医師の指示等により対応が異なる場合があるため、必ず医師の指示および院内ルールを優先してください。

1. はじめに

本ツール群は、インストール不要・ネット接続不要で動くブラウザアプリです。新人教育、手順の確認、プログラマー操作の練習に使えます。実機の代替・診断用途ではありません。

動作環境・基本操作

  • Chrome / Edge / Safari などの一般的なブラウザで HTMLファイルを開くだけ。参考文献リンクを開くとき以外はオフラインで動作します。
  • 右上の ☀ ライト / ☾ ダーク でテーマを切替えできます。
  • 練習履歴・設定は 使用したブラウザ内（localStorage）に保存されます。別PC・別ブラウザには引き継がれません。
  • 入口は CIED check Lab。ここから各ツールを開きます。

2. ツール構成（目的 × 対象デバイス）

入口の Lab は「学ぶ」と「試す（操作シミュレーター）」の2系統に分かれ、試す側は対象デバイス別に選びます。

4. CIED check Arena（基本PM 操作シミュレーター）

症例ごとに Programmer（プログラマー）を操作し、波高値・閾値のチェックを練習・採点するツールです。波高値は自己波、閾値はペーシングが出ている条件で測定するのが基本です。

画面の見方

  • ECG + Marker：心電図とマーカー（下表）を表示。
  • Programmer：下限レート・SAV/PAV・A/V出力・A/V感度・DDD/VVIモードを操作。各行に初期値からの差を表示。
  • Measurements：4項目（A波高値・V波高値・A閾値・V閾値）を「記録」または「測定不可」で判断。先頭に デフォルト設定（設定を症例開始時へ戻す）、末尾に テスト終了。
マーカー意味測定対象
AS自己P波を感知A波高値
AP心房ペーシングA閾値
VS自己R波を感知V波高値
VP心室ペーシングV閾値

基本の流れ

  1. 上部の「トレーニング症例」を選ぶ。
  2. Programmerで設定を変え、Markerを目的の状態（AS/AP/VS/VP）にする。
  3. 波高値は自己波（AS/VS）中に「記録」、閾値はペーシング（AP/VP）を出力低下させて捕捉/脱落（LOC）境界を確認し「記録」。出ない症例は「測定不可」を判断。
  4. 測定後は デフォルト設定で設定を戻す。
  5. テスト終了で採点。

採点ルール

  • 100点満点 / 合格80点 / 全4項目クリア。
  • 無関係な動作（目的外の設定・モード変更）は −10点（同方向の連打は初回のみ）。
  • 誤判定（出るのに測定不可など）も減点。正しい「測定不可」は加点。
  • VVIへ切替えてV波高値を確認した後は、DDDへ戻すまで次の測定に進めません（実臨床でも一時的なVVIのまま放置しないため）。

便利機能

  • 🔊 / 🔈 音（心電図音・効果音）ON/OFF、ヒント ON/OFF。
  • DDD / VVI モード切替。下限レートは最大200 ppmまで。
  • キーボード操作にも対応（各操作ボタンにショートカット表示）。

より詳しい操作は Arena 操作マニュアル（HTML） を参照してください。

5. CIED check CRT

CRT-P / CRT-D の同期・最適化を学ぶシミュレーターです。症例を選び、AV/VV delay・LV offset・BiV capture・LV出力などを調整し、同期スコアと推定QRS幅の変化を確認します。

テストモード

  • テスト開始→各項目を最適化→テスト終了で採点（5項目・各18点＋全達成ボーナス10＝100点、合格80点）。
  • 主な最適化項目：AV delay / VV delay / LV offset / BiV capture / LV出力。
  • MPP（マルチポイントペーシング）の有効性は不確実なため、本ツールでは 「検討」扱いで採点には含めません。

6. CIED check ICD

ICD / CRT-D の検出ゾーンと治療設定を学ぶシミュレーターです。症例の入力レートとEGM文脈に対し、ゾーン・SVT判別・ATP・ショックの設定を判断します。

テストモード

  • テスト開始→各項目を最適化→テスト終了で採点（5項目・各18点＋全達成ボーナス10＝100点、合格80点）。
  • 主な最適化項目：Monitor / VT / VF ゾーン、Detection（検出時間）、治療設定（SVT判別・ATP Burst/Ramp・ショック）。
  • 不適切作動を避ける設計（高めの治療ゾーン・長めの検出・SVT判別・ATP先行）の考え方を練習できます。
