2. 基本ロジック

Arenaでは、測りたい項目に合わせて作動状態を作ります。波高値は自己波、閾値はペーシングが出ている条件で評価します。

4種類のチェック

項目 記録できる条件 条件を作る代表操作
A波高値 AS表示中。自己P波が出ている。 AP主体なら下限レートを下げ、自己P波を出す。
V波高値 VS表示中。自己R波が出ている。 DDDではSAV/PAVを延長。VVIでは下限レートを下げる。
A閾値 AP表示中。Aペーシング捕捉を観察できる。 AS主体なら下限レートを上げ、APを出す。
V閾値 VP表示中。Vペーシング捕捉を観察できる。 DDDではSAV/PAVを短縮。VVIでは下限レートを上げる。

基本の進め方

1. 症例を選ぶ
疾患、モード、現在のMarker、目標を確認します。
2. ECGとMarkerを見る
今がAS/AP/VS/VPのどれか、SensingとCaptureが成立しているかを見ます。
3. Programmerで条件を作る
下限レート、SAV/PAV、出力を変更し、Marker変化を確認します。差分バッジ（+/−）で初期値からのズレが分かります。
4. Measurementsで記録する
条件がそろった項目だけ記録します。閾値は 開始 → 出力低下 → LOC確認 → 記録 の順です。各項目の状態は番号バッジで確認できます。
5. 初期設定へ戻して「テスト終了」
一時変更を初期値へ戻し、テスト終了ボタンを押します。設定が戻っていないとペナルティが入ります。
AV delay の使い分け
AS（自己P波あり）の時は SAV、AP（心房ペーシング）の時は PAV が作動に効きます。 AS中にPAVを動かしても作動は変わらず、誤操作扱いで減点されます。

Hands-on

3. 代表操作パターン

状況 やりたいこと 操作の考え方
DDD / AP-VS A波高値を測る 下限レートを下げてAPからASへ変える。ASになったらA波高値を記録。
DDD / AS-VP V波高値を測る SAVを延長し、AS-VPからAS-VSへ変わるか確認。VSが出ればR波を記録。
DDD / AP-VP V波高値を測る PAVを延長または下限レートを下げて自己伝導を確認。自己R波が出なければ評価困難。
DDD / AS-VS V閾値を測る SAVを短縮してVPを出す。V閾値を開始し、V出力を下げてLOCを確認して記録。
VVI / VP V波高値を測る 下限レートを自己心拍より下げ、VSが出るか確認。出ればR波を記録。
VVI / VS V閾値を測る 下限レートを上げてVPを出す。V出力を下げて捕捉/脱落境界を確認。

閾値測定の操作

  1. APまたはVPが出ていることを確認します。
  2. A閾値またはV閾値の 開始 を押します。
  3. A出力またはV出力を少しずつ下げます。
  4. ECGとMarkerで 捕捉あり、脱落(LOC) の境界を確認します。
  5. 境界確認後に 閾値記録 を押します。
  6. 出力を安全域へ戻し、必要なら 初期設定へ戻す を押します。

「VS が出ない」を確認する手順（CAVB系）

AS主体で自己R波が出ない症例（完全房室ブロック等）では、SAVを 280ms以上 に延長し、それでもVSが出ないことを確認すると VS出ない確認 ステップが自動完了します。 その後 無記録で終了（V波高値は評価困難）を選びます。

V捕捉消失・自己VSなしの警告
V閾値測定中または通常測定中に、VP捕捉が外れて自己VS/逃拍が期待できない症例では警告が出ます。 教育用表示では、心室停止と脈圧消失を想定し、直ちにV出力を安全域へ戻す判断を学習します。

評価困難を選ぶ時

V波高値は自己R波が必要です。AV delay延長や下限レート低下を行ってもVSが出ない症例では、 無理にR波測定へ進まず 無記録で終了 を選びます。先に「VS出ない確認」を済ませると確実です。

Scoring

4. 採点ルール

満点100点
合格基準80点
誤操作1回−10点

得点・減点の内訳

区分 イベント 点数
加点 各ステップ正解（条件作成・記録など） 合計 90点（症例ごとに配分）
テスト終了時の完了ボーナス +10点
減点 条件未成立で記録、目的外の設定変更、操作方向ミス −10点
進捗を後退させる操作（VSが出た後にSAV短縮など） −10点
未復帰（初期設定に戻さず）でテスト終了 −10点
未完了状態でテスト終了（初回のみ） −10点

減点の優しい仕様（debounce）

  • 同じ誤操作の連打は1回のみ減点。たとえばSAVを誤った方向に5回連続で動かしても −10点 一度きりです。
  • 正しい操作・別キーの操作・ステップ完了でリセットされ、次の誤操作は再度減点対象になります。
  • 「未完了で終了」連打も初回のみ減点。
  • 記録系ステップ中の同方向への追加調整（VS確認後にさらにSAV延長など）は警告のみ・減点なし。逆方向で進捗を壊した時のみ減点。

合否判定の考え方

  • 満点は 各ステップ加点 90点 + ボーナス 10点 = 100点。
  • 誤操作 2回まで（80点）で合格、3回目以降は不合格になります。
  • 未復帰でテスト終了するとボーナスとペナルティが相殺され、表示される delta（差分）と実際のスコア変化が一致します。
  • raw（生スコア）が0未満になる事もあります。表示はそのまま負の数で出ます。

効果音

イベント 音
ステップクリア 軽快な2音アルペジオ（A5 → E6）
テスト終了 / 合格 「激しい勝利ファンファーレ」（上昇アルペジオ＋連続コード＋低音＋打楽器ノイズ）
テスト終了 / 不合格 柔らかい降下音（A4 → E4）

Practice

5. 練習のコツ・よくあるミス

ヒント ON / OFF の使い分け

設定 使い方
ヒント ON（初期） 次に見るべき項目や推奨操作が表示されます。初回学習や解説用に使います。
ヒント OFF 具体的な操作手順が伏せられます。採点付きの実践練習に使います。

よくあるミス

ミス 確認ポイント
VP中にV波高値を記録する V波高値はVS表示中に記録します。まず自己R波を出します。
VS中にV閾値を開始する V閾値はVP表示中に開始します。VP条件を作ってから開始します。
AP中にA波高値を記録する A波高値はAS表示中に記録します。下限レートを下げて自己P波を出します。
AS中にPAVを延ばす（無効操作） AS時はSAVが作動に効きます。PAVは AP 時のみ意味があります。
LOC確認前に閾値記録する 出力を下げ、捕捉/脱落境界を見てから記録します。
測定後に設定を戻さない テスト終了前に 初期設定へ戻す を押し、Mode、LRL、AV delay、出力を確認します。
VS確認後にSAVを短くする 進捗が後退する操作は減点。記録ボタンに進みましょう。

おすすめ練習順

  1. ヒント ON で1症例を完走（操作・採点の流れを把握）。
  2. 同じ症例を ヒント OFF で再挑戦（手順を自分で組み立てる）。
  3. ランダム症例 で実践練習（履歴で得点推移を確認）。
  4. 合格できたら、誤操作0回 100点の完全クリアを目指す。
迷った時は「波高値は自己波、閾値はペーシング」を起点にMarkerを見直してください。 AS/AP/VS/VPのどの状態にあるかを確認すれば、次の操作の方向が見えてきます。
練習履歴
画面下部に直近10件の症例・得点・誤操作・合否が記録されます。同じ症例を繰り返して上達を確認できます。履歴は「履歴を消去」で全削除できます（ブラウザに保存されているため、別PCには引き継がれません）。
