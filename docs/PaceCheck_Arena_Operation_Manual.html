Core Logic

2. 基本ロジック

測りたい項目に合わせて作動状態を作る。波高値は自己波、閾値はペーシングが出ている条件で評価する。

AS自己P波 / A波高値
APA pace / A閾値
VS自己R波 / V波高値
VPV pace / V閾値
項目記録できる条件条件を作る代表操作
A波高値AS表示中。自己P波あり。AP主体なら下限レートを下げ、自己P波を出す。
V波高値VS表示中。自己R波あり。DDDではSAV/PAVを延長、VVIでは下限レートを下げる。
A閾値AP表示中。A捕捉を観察。AS主体なら下限レートを上げ、APを出す。
V閾値VP表示中。V捕捉を観察。DDDではSAV/PAVを短縮、VVIでは下限レートを上げる。

基本の進め方

① 症例を選び、ECG・Markerで現在の作動を把握。
② Programmerで条件を作る — 下限レート/SAV/PAV/出力を変更しMarker変化を確認。
③ Measurementsで記録 — 条件成立項目のみ。閾値は 開始 → 出力低下 → LOC確認 → 記録。
④ 初期設定へ戻して「テスト終了」 — 未復帰だとペナルティ。全4項目で採点。

Hands-on

3. 代表操作パターン

状況やりたいこと操作の考え方
DDD / AP-VSA波高値を測る下限レートを下げてAPからASへ。ASになったらA波高値を記録。
DDD / AS-VPV波高値を測るSAVを延長し、AS-VPからAS-VSへ変わるか確認。VSが出ればR波を記録。
DDD / AP-VPV波高値を測るPAV延長または下限レート低下で自己伝導を確認。自己R波が出なければ評価困難。
DDD / AS-VSV閾値を測るSAVを短縮してVPを出す。V閾値開始 → V出力を下げてLOC確認 → 記録。
VVI / VPV波高値を測る下限レートを自己心拍より下げ、VSが出るか確認。出ればR波を記録。
VVI / VSV閾値を測る下限レートを上げてVPを出す。V出力を下げて捕捉/脱落境界を確認。

閾値測定の手順

  1. APまたはVPが出ていることを確認。
  2. A閾値またはV閾値の 開始 を押す。
  3. A出力またはV出力を少しずつ下げる。
  4. ECGとMarkerで 捕捉あり / 脱落(LOC) の境界を確認。
  5. 境界確認後に 閾値記録 を押す。
  6. 出力を安全域へ戻し、必要なら 初期設定へ戻す。

「VS が出ない」確認（CAVB系）

自己R波が出ない症例（完全房室ブロック等）では、SAVを 280ms以上 に延長してもVSが出ないことを確認すると VS出ない確認 ステップが自動完了する。その後 無記録で終了（V波高値は評価困難）を選ぶ。

評価困難を選ぶ時

AV delay延長や下限レート低下を行ってもVSが出ない症例では、無理にR波測定へ進まず 無記録で終了 を選ぶ。先に「VS出ない確認」を済ませると確実。

V捕捉消失・自己VSなしの警告 — VP捕捉が外れて自己VS/逃拍が期待できない症例では警告が出ます。教育用表示では、心室停止と脈圧消失を想定し、直ちにV出力を安全域へ戻す判断を学習します。

Scoring

4. 採点ルール

満点100点
合格基準80点
誤操作1回−10点

得点・減点の内訳

区分イベント点数
加点各ステップ正解合計 90点
テスト終了時の完了ボーナス+10点
減点条件未成立記録、目的外操作、方向ミス−10点
進捗を後退させる操作−10点
未復帰でテスト終了−10点
未完了状態でテスト終了（初回のみ）−10点

減点の優しい仕様（debounce）

  • 同じ誤操作の連打は 1回のみ減点。
  • 正しい操作・別キー・ステップ完了でリセット。
  • 「未完了で終了」連打も初回のみ減点。
  • 記録系ステップ中の 同方向への追加調整は警告のみ・減点なし、逆方向で進捗を壊した時のみ減点。

合否

  • 満点 = ステップ加点90 + ボーナス10。
  • 誤操作 2回まで（80点）で合格、3回目以降不合格。
  • 未復帰時はボーナスとペナルティが相殺し delta 表示と実差分が一致。

Practice

5. 練習のコツ・よくあるミス

ヒント ON / OFF の使い分け

設定使い方
ヒント ON（初期）次に見るべき項目や推奨操作が表示される。初回学習や解説用。
ヒント OFF具体的な操作手順が伏せられる。採点付き実践練習用。

効果音

イベント音
ステップクリア軽快な2音アルペジオ（A5→E6）
合格激しい勝利ファンファーレ（上昇アルペジオ＋連続コード＋低音＋打楽器ノイズ）
不合格柔らかい降下音（A4→E4）

おすすめ練習順

  1. ヒント ON で1症例を完走（流れを把握）
  2. 同じ症例を ヒント OFF で再挑戦
  3. ランダム症例 で実践練習（履歴で得点推移を確認）
  4. 合格後、誤操作0回 100点の完全クリアを目指す

よくあるミス

ミス確認ポイント
VP中にV波高値を記録VS表示中に記録。まず自己R波を出す。
VS中にV閾値を開始V閾値はVP表示中に開始。VP条件を作る。
AP中にA波高値を記録AS表示中に記録。下限レートを下げて自己P波を出す。
AS中にPAVを延ばすAS時はSAVが作動に効く。PAVはAP時のみ。
LOC確認前に閾値記録出力を下げ、捕捉/脱落境界を見てから記録。
測定後に設定を戻さないテスト終了前に「初期設定へ戻す」。Mode、LRL、AV delay、出力を確認。
VS確認後にSAVを短くする進捗が後退する操作は減点。記録に進む。
迷った時は 「波高値は自己波、閾値はペーシング」 を起点にMarkerを見直す。AS/AP/VS/VPのどの状態にあるかを確認すれば、次の操作の方向が見えてくる。
練習履歴 — 画面下部に直近10件の症例・得点・誤操作・合否が記録される。同じ症例を繰り返して上達を確認。「履歴を消去」で全削除（ブラウザ保存のため別PCには引き継がれない）。
